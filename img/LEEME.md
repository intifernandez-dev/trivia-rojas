# Imágenes requeridas

Coloca las siguientes imágenes en esta carpeta:

## Imágenes de interfaz:
- **auris.png** - Logo/mascota de la trivia
- **puntos.png** - Icono de puntos
- **user.png** - Icono de usuario
- **medalla.png** - Medalla de victoria

## Imágenes de categorías:
- **general.png** - Ícono para categoría HTML
- **deportes.png** - Ícono para categoría Diseño Web
- **peliculas.png** - Ícono para categoría Wordpress
- **musica.png** - Ícono para categoría Cultura Morón

## Formato recomendado:
- PNG con fondo transparente
- Tamaño: 200x200px para categorías
- Tamaño: 80x80px para iconos de interfaz
